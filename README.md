# Heart-Disease-Prediction
Machine learning-based heart disease prediction using multiple classification models including Logistic Regression, Random Forest, SVM, and KNN. Includes data preprocessing, EDA, and model evaluation with accuracy and ROC-AUC metrics.

This project applies multiple machine learning models to predict the presence of heart disease using patient medical data.

## 📌 Overview

Heart disease is a major global health concern. Early prediction can help in timely diagnosis and treatment. This project uses machine learning techniques to classify whether a patient is at risk.

## 🎯 Objective

* Predict heart disease using medical features
* Compare multiple machine learning models
* Evaluate performance using standard metrics

## ⚙️ Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)

## 🔍 Methodology

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature selection
* Model training and comparison

## 📈 Results

* Best Accuracy: ~88.9%
* ROC-AUC: ~0.90

## 📊 Dataset

UCI Heart Disease Dataset (Kaggle)
Dataset not included due to size limitations.
